# Collection Development Policy Files
* These policy files have been developed alongside all the other material within this repository. 
* There are two formats:
  * A simplified summary that will hopefully change very little after its initial drafting 
  * The full policy in bulleted list format which be reviewed and changed annually.
* There are three formats for these files: PDF for showing, Word for formatting, and Markdown for ease of editing online.
* The collection policy contains instructions for its use and best practices.
