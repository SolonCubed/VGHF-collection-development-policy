# Interview Data
These files contain interview data from five media libraries and archives:

- The National Film and Sound Archive of Austraila (February 19th, 2026)
- Blizzard Archive (January 21st, 2026)
- Embracer Games Archive (May 28th, 2026)
- The Media History Digital Library (February 20th, 2026)
- The Netherlands Institute of Sound and Video (April 20th, 2026)

Each interview contains the following file formats:

- Audio: .OPUS 
- Text: .TXT
- Subtitles: .VTT

My Apologies:
I don't believe I'll ever get to cleaning up the text/subtitles that were generated by simple Voice-To-Text algorithm. I was also sick when I interviewed Blizzard Archive so my audio is a bit rough to listen to. Sorry.

Video versions of these interviews can be made available by request to the original owner, Solon, but not all things in those videos should be shared publicly yet so give me heads up on what you want to see and we can make it work.
